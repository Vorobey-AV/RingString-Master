# RingString Master v2.0

Самая удобная и качественная программа для создания стринг-арта из фото в 2025 году!

[![Preview](preview.jpg)](preview.jpg)

## Возможности
- Загрузка любого фото
- Автоудаление фона (rembg)
- Drag & Drop, яркость, контраст, масштаб
- Умный автоподбор параметров
- Генерация с анимацией или в 10× быстрее
- Пошаговый плеер-инструкция
- Экспорт: TXT • JSON • CSV

## Установка и запуск (одна команда)

```bash
pip install pillow numpy rembg[cli]
python ringstring_master.py
